# Simply-CHIP8
## WIP Chip-8 Emulator

This is my first attempt at a major project written in C.

### Currently Working:
- Reading rom files
- Memory (and storing roms into memory)
- Certain CPU opcodes (isolated)

### In Progress:
- Full CPU opcode implementations
- Working CPU cycling / timers
- Stack
- Display
- Keypad Input
- Sound
