# Simply-CHIP8
## WIP Chip-8 Emulator

This is my first attempt at a major project written in C.

### Currently Working:
- Reading rom files
- Memory (and storing roms into memory)
- CPU Cycles
- All CPU Opcodes
- Display
- Keypad Input

### In Progress:
- Delay timer
- Sound timer

### Future Plans:
- Full rewrite
- Debug window
